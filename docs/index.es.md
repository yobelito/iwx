# IWX – Índice de documentación del protocolo Index Wire eXchange

- **Especificaciones del protocolo**: consulta `specs/IWX-SPEC-v1.0.md`, `specs/IWX-SECURE-v1.0.md`, `specs/TRANSPORT-PROFILES.md` y `specs/SCHEMA-SPEC.md`.
- **Estado del repositorio y hoja de ruta**: `docs/STATUS.md` y `docs/STATUS.es.md`.
- **README principal (inglés)**: `README.md` con instalación, uso, puntos destacados de la API y roadmap.
- **README en español**: `README.es.md` para un inicio rápido localizado.
- **Esquemas**: ejemplos de referencia en `schemas/` (por ejemplo, `schemas/sale/schema.json`).
- **Bibliotecas**: implementaciones en JavaScript, Python y PHP en `libs/` con helpers plain, secure y binarios.
- **Manifiesto frontend/backend**: guía en `docs/frontend-backend-ejemplo.md` (ES) y `docs/frontend-backend-example.md` (EN).
- **Gestión de esquemas**: recomendaciones en `docs/schema-management.md` (EN) y `docs/schema-management.es.md` (ES).
